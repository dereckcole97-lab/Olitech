# Olitech
Phone Repair Shop
## Description
This project is a modern and responsive for a phone repair shop. Built to showcase pricing, services available and contacts informations.
The website is built to provide easy access for customers and easy way to explore the services, contacts (to get in touch)
Page entirely in italian language as the person who owns the shot is from there and its location is in Italy.
## Features
- Designed for both desktop and mobile screen
- availability provided with working daily hours and day/days off
- type of services provided
- timing of the services (waiting time)
- location
## Technologiees used
- HTML5
- CSS3
- JAVASCRIPT
- Tailwind CSS
